# Цветовая палитра проекта

## Основная палитра для строительной компании "Честные дома"

### Основные цвета

- **Основной акцентный цвет (терракотовый/кирпичный)**: `#C85A3A`
  - Используется для: кнопок, ссылок, заголовков, акцентов
  - Tailwind класс: `bg-[#C85A3A]`, `text-[#C85A3A]`, `border-[#C85A3A]`

- **Hover состояние**: `#B04A2E`
  - Используется для: hover эффектов на кнопках и ссылках
  - Tailwind класс: `hover:bg-[#B04A2E]`

### Нейтральные цвета

- **Текст**: `#111827` (gray-900)
  - Используется для: основного текста на белом фоне
  - Tailwind класс: `text-gray-900`

- **Фон**: `#FFFFFF` (белый)
  - Используется для: основного фона страниц
  - Tailwind класс: `bg-white`

- **Границы**: `#E5E7EB` (gray-200)
  - Используется для: границ карточек и элементов
  - Tailwind класс: `border-gray-200`

### Использование

```jsx
// Кнопки
<button className="bg-[#C85A3A] text-white hover:bg-[#B04A2E]">
  Кнопка
</button>

// Ссылки и заголовки
<h2 className="text-[#C85A3A]">Заголовок</h2>
<Link className="text-[#C85A3A] hover:opacity-80">Ссылка</Link>

// Границы
<div className="border border-[#C85A3A]">Элемент</div>

// Focus состояния
<input className="focus:ring-[#C85A3A]" />
```

### Обоснование выбора

Терракотовый/кирпичный цвет выбран потому что:
- Ассоциируется со строительством и кирпичом
- Вызывает доверие и надежность
- Теплый и приятный для восприятия
- Хорошо сочетается с белым фоном
- Подходит для строительной компании
