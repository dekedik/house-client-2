## Формат изображений:
- Рекомендуемый формат: JPG или PNG
- Рекомендуемое разрешение: минимум 800x600 пикселей
- Оптимизируйте изображения для веб-использования

## Заглушка:
Если изображение не найдено, будет использоваться `placeholder.svg`. 
Этот файл уже создан и будет использоваться автоматически при отсутствии изображений.
